# Glossar – Bau & Stil

> Wiederkehrende Fachbegriffe rund um Future Planning. Wächst mit dem Projekt.
> Kontext: [[Future-Planning-Vision]].

| Begriff | Bedeutung |
|---|---|
| **SIA** | Schweizerischer Ingenieur- und Architektenverein; gibt Normen heraus (z.B. zu Massen/Abständen), an denen sich die Planung orientiert. |
| **BKP** | Baukostenplan – schweizerisches Gliederungsschema für Baukosten. „BKP zweistellig" = Kostengliederung auf zwei Ebenen. |
| **KV** | Kostenvoranschlag. Im Produkt ein **Richtwert-KV** (grobe, automatisierte Kostenschätzung). |
| **Gewerk** | Abgegrenzte Handwerks-/Bauleistung (z.B. Maler, Sanitär, Elektro). |
| **Stilvektor / Stilprofil** | Numerische Repräsentation der Stilvorlieben über 8 Achsen (aus Modul 01). |
| **Coverage-Grid / -Score** | Raster zur Live-Anzeige, wie vollständig ein Raum gescannt ist. |
| **Fixpunkt** | Nicht frei platzierbarer Punkt im Raum (z.B. Anschluss, fester Einbau), den die Planung berücksichtigen muss. |
| **Harte Regeln** | Unverhandelbare Planungsregeln (Normen, Mindestabstände, Kollisionen, Anschlüsse). |
| **Weiche Regeln** | Stil-/Präferenz-basierte Optimierungsziele aus dem Stilprofil. |
| **GU** | Generalunternehmer – übernimmt Bau/Umbau gesamthaft; im Ökosystem B2B-Partner. |
| **Bemusterung** | Auswahl von Materialien/Ausstattung durch Käufer im (Neu-)Bau. |
| **TAM / SAM / SOM** | Markt­grössen: Total/erreichbarer/realistisch erzielbarer Markt. |
| **Freemium / Affiliate** | Gratis-Basisnutzung mit Zusatzkäufen / Provision auf vermittelte Käufe. |

## BIM- & Daten-Begriffe (langfristige Vision)
| Begriff | Bedeutung |
|---|---|
| **BIM** | Building Information Modeling – modellbasiertes Arbeiten mit Bauwerksdaten. |
| **IFC** | Industry Foundation Classes – offenes Austauschformat für BIM-Daten. |
| **CDE** | Common Data Environment – gemeinsame Datenumgebung für ein Bauprojekt. |
| **DWG** | CAD-Dateiformat (AutoCAD) für 2D-/3D-Pläne. |
| **FM** | Facility Management – Betrieb/Bewirtschaftung der Immobilie. |

## Technische Begriffe (zur Vertiefung im neuen Konzept)
| Begriff | Bedeutung |
|---|---|
| **Fotogrammetrie** | 3D-Rekonstruktion aus überlappenden Fotos/Frames. |
| **VIO / SLAM** | Visual-Inertial Odometry / Simultaneous Localization and Mapping – Positions- und Karten-Schätzung aus Kamera (+ Sensoren) in Echtzeit. |
| **Dense Reconstruction** | Dichte Punktwolke/Mesh aus vielen Ansichten (genau, aber rechenintensiv – z.B. COLMAP/OpenMVS). |
| **Plane-/Objekt-Segmentierung** | Erkennen von Flächen (Wand/Boden/Decke) bzw. Objekten im 3D-Raum. |
| **glTF 2.0** | Offenes Austauschformat für 3D-Szenen/Modelle. |
