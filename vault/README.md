# vault/ – Das digitale Brain 🧠

Dies ist der **Obsidian-Vault** von FP_Kopf. Hier liegt **ausschließlich
fachliches Wissen**. Wie gearbeitet wird, steht im Repo-Root
([`../CLAUDE.md`](../CLAUDE.md)) – nicht hier.

## Öffnen in Obsidian
„Open folder as vault" → diesen `vault/`-Ordner wählen. Einstieg:
[`00_Start/Start.md`](00_Start/Start.md).

## Ordner

| Ordner | Inhalt |
|---|---|
| `00_Start/` | Einstieg / Map of Content (MOC) |
| `10_Learnings/` | Erkenntnisse, Erfahrungswerte |
| `20_Architektur/` | Komponenten, Datenflüsse, Systemstruktur |
| `30_Entscheidungen/` | Entscheidungs-Records (ADR-Stil) |
| `40_Produktkonzepte/` | Features, Produktideen |
| `50_Umsetzung/` | Umsetzungswege, How-tos, technische Details |
| `60_Quellen/` | Externe Quellen, Recherche |
| `90_Templates/` | Vorlagen für neue Notizen |
| `99_Archiv/` | Abgelöste fachliche Notizen |

## Konventionen
- Atomare, verlinkte Notizen mit YAML-Frontmatter (siehe `90_Templates/`).
- Neue Notizen im MOC (`00_Start/Start.md`) verlinken.
- Nichts löschen – Abgelöstes nach `99_Archiv/` verschieben.
- Details: Skill `brain-eintrag` in `../.claude/skills/`.
